About This Project

Alpha One Index is an independent, open-source research initiative that aggregates and indexes critical data across the AI ecosystem. All data is sourced from public repositories, official documentation, and peer-reviewed research. Automated pipelines refresh each index on its own schedule.

Every dataset and methodology is transparent and available on GitHub.
